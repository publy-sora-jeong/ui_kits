TAB UI

                

                    
    
                        
  • role="tab" 버튼이 탭버튼임을 명시합니다 
    • 
                        
  • aria-selected 값으로 해당 버튼이 선택 된 상태인지 판별합니다 
    • 
                        
  • 탭리스트의 aria-controls="아이디" 탭패널의 아이디값은 동일하게 지정하여 서로 연계되어있다는 것을 명시합니다 
    • 
                        
  • 선택 되지 않은 버튼의 tabindex 값은 -1로 설정하여 키보드 탭으로 접근이 되지 않게 합니다.
    • 
                        
  • role="tabpanel" 로 해당 div 가 연결된 tab button이 있는 tab panel이라는 것을 명시합니다.
    • 
                        
  • 현재 tab panel과 연결된 tab button의 id 를 aria-labelledby="tab button id" 로 연결합니다.
    • 
                        
  • 활성화 되지 않은 tab panel은 스크린 리더기가 읽을 수 없게 hidden="false" 속성을 넣어 숨겨줍니다.
    • 
                        
  • role="tabpanel" 속성은 ul 요소에 사용할 없습니다.

Swiper

                

                    
    
                        
  • autoplay인경우 초점이 있으면 슬라이드 멈춤 , 키보드사용자를위한 제어버튼(재생, 정지)은
                                슬라이드 시작전에 배치
    • 
                        
  • aria-current 속성으로 현재 표시중인 배너의 인디케이터임을 시각효과와 함꼐 명시
    • 
                        
  • 이전, 다음 아이템 버튼을 눌렀을 때 aria-live를 사용하여 화면의 변화(내용)를 알려야 한다.
    • 
                        
  • aria-hidden="true'와 tabindex="-1"를 통해 현재 눈에 보이는 이미지 아이템 외의 모든 아이템을 숨겨야 한다.
    • 
                        
  • 사용자 요청에 의한 캐러셀 전환이 아니라면 aria-live로 스크린리더에 알림을 주어서는 안 된다.
